# Toyota Stock Market Analysis (NYSE: TM) 🚗📈

This project contains a **comprehensive analysis of Toyota stock prices** over 40+ years using **Excel, Python, PostgreSQL, and Power BI**.  

It demonstrates end-to-end financial data analytics skills: from **data cleaning**, **feature engineering**, **SQL queries**, **visualizations**, to **forecasting**.

---

## Features
- **Data Cleaning & Preparation**: Handling missing values, type conversions, setting date index.
- **Feature Engineering**: Moving averages (50-day & 200-day), daily returns, volatility, profit/loss classification.
- **Financial Metrics**: CAGR, Sharpe Ratio, Drawdown, Risk vs Return.
- **SQL Queries for Insights**: Maximum price, minimum price, yearly returns, volume analysis.
- **Power BI Dashboard**: Interactive charts, KPIs, correlation heatmap.
- **Time Series Forecasting**: Trend-based future price predictions.

2. Run SQL scripts in PostgreSQL from `3_SQL/` to extract insights.
3. Open Power BI dashboard from `4_PowerBI/` to explore interactive visualizations.
